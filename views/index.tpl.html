telepush

Send Telegram push notifications easily via HTTP

Usage

Step 1: Get a token

Start a chat with TelepushBot (or your own bot) and type /start to obtain a recipient token.

Step 2: Send your message

curl -XPOST \
   -d '{ "text": "Hello there" }' \
   api/messages/<recipient>
